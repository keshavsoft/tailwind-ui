🛠 Available Commands

CLI workflow to generate APIs step by step

1. StartEndPoint

Creates the base API folder structure.

npx @keshavsoft/kschema-api-gen StartEndPoint

2. AddSubRoute

Creates API version folders like V1 and V2.

cd Api npx @keshavsoft/kschema-api-gen AddSubRoute

3. AddTableName

Create module/table folders such as Accounts or Users.

cd V1 npx @keshavsoft/kschema-api-gen AddTableName

4. ShowAll

Generate GET endpoint architecture for fetching records.

cd KSTable1 npx @keshavsoft/kschema-api-gen ShowAll

5. CreateApi

Automatically hooks generated endpoints into Express routes.

npx @keshavsoft/kschema-api-gen CreateApi Api/V1/KSTable1/ShowAll

6. InsertApi

Create POST/Insert endpoint structures instantly.

npx @keshavsoft/kschema-api-gen InsertApi Api/V1/KSTable1/Insert
